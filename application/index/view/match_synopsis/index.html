2019中国篮球公开赛

中国篮球公开赛(CHINA BASKETBALL OPEN)是由中国篮球协会主办,各地市级篮球协会承办,为落实全民健身国家战略、倡导文明健康生活方式、促进全民健身与全民健康深度融合,并进一步推动我国篮球运动的发展,加快篮球文化在中国的普及和宣导,而推出的一项由中国3亿篮球爱好者参与的全国性大型公益篮球赛事活动平台。

中国篮球公开赛定位于公益性的篮球品牌,是一个扎根于基层的草根性群体赛事活动平台,其在我国群众体育发展中定位于“探索之路”——赛事始终坚持“市场化运作”的运营方针,在我国举国体制群体发展的背景下,力求探索出一条自给自足的良性循环模式,实现群众体育项目“协会主导,社会办赛”的可持续发展。

中国篮球公开赛是在我国篮球运动发展中定位于“基石”、“基础”、“草根”,她在吸引不同地域、不同民族、不同行业、不同年龄阶段篮球爱好者参与的同时,自然的完成了篮球运动普及与宣导的过程,为我国篮球运动的快速发展奠定稳固的“基石”。
