关于2019中国篮球公开赛

比赛规则

中国篮球公开赛(CHINA BASKETBALL OPEN，简称:CBO)是由中国篮球协会主办，各地市级篮球协会承办，为落实全民健身国家战略、倡导文明，健康生活方式、促进全民健身与全民健康深度融合，并进一步推动我国篮球运动的发展，加快篮球文化在中国的普及和宣导，而推出的一项由中国3亿篮球爱好者参与的全国性大型公益篮球赛事活动平台。
