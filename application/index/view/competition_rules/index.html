比赛时间

  • 城市预选赛

    比赛时间为2019年8月10日至2019年9月8日,共5个周末,10个比赛日,各承办城市按照本城市实际情况选择相应的比赛日完成比赛。

  • 大区赛

    比赛时间为2019年8月10日至2019年9月8日,共5个周末,10个比赛日,各承办城市按照本城市实际情况选择相应的比赛日完成比赛。

  • 总决赛

    比赛时间为2019年8月10日至2019年9月8日,共5个周末,10个比赛日,各承办城市按照本城市实际情况选择相应的比赛日完成比赛。

  • 女子冠军赛

    为赛会制比赛,比赛时间为2019年10月16日至2019年10月23日举行举行(具体时间另行通知),各赛区城市申办举行,由各省推荐女子篮球队参加(球队16支,即:每个省或自治区报名一支参赛队,共涉及16个省),比赛为时5天。

城市预选赛

各承办城市定向招募16支球队,赛前进行抽签分组,采用小组单循环+淘汰赛的赛制;小组单循环阶段采用积分赛制,累计积分:胜积2分负积1分,弃权球队得0分,弃权场次达两场即取消参赛资格;小组积分第一进入淘汰赛阶段比赛前三节毛时,第四节最后2分钟净时,全场比赛双方运动队暂停时停表,其余时间不停表
